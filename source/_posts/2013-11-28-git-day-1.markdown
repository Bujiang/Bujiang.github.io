添加了一个readme 文件后再用 git status 就有提示有添加新的文件


添加文件可以有多种方式 

如 git add readme.txt bujiang.txt  一一列举需要添加的文件

git add --all  添加所有

git add *.txt  添加所有txt文件

git add css/  添加整个css文件夹
                   

想知道添加了什么  可以使用 git log来查询历史记录
